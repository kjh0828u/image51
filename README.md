# Image51 - 스마트 클라이언트 AI 이미지 처리기 🖼️✨

Image51은 서버나 클라우드를 거치지 않고, 사용자 브라우저(Client-Side)의 자원을 활용하여 이미지를 변환, 압축, 추출하는 웹 & 데스크톱 툴입니다.

## 🚀 주요 기능

### 1. 100% 로컬 연산 및 초고강도 프라이버시 보호
외부 서버 API나 클라우드 전송 과정이 없습니다. 사용자 컴퓨터 내 브라우저에서 연산하고 처리합니다.

### 2. AI 기반 정리 배경 제거 (누끼)
브라우저용 머신러닝 라이브러리(`Transformers.js`)를 통해 원클릭으로 피사체 주변 배경을 투명하게 날려버립니다.
* **디테일 옵션 제공:** 경계면 부드럽게(Alpha Matting), 경계선 침식(Erode), 가짜 그림자 및 사물 내부 배경파임 등 세세한 영역까지 슬라이더로 조절하여 포토샵급 퀄리티를 유지합니다.

### 3. 지능형 빈 여백 자동 자르기 (Auto Crop)
이미지 픽셀 단위로 의미 있는 영역의 상하좌우 경계를 스스로 스캔/탐지해 불필요한 투명 알파 스페이스를 자동으로 잘라냅니다. (여백 Margin 값 조절 가능)

### 4. 강력한 브라우저 포지션: 다이렉트 폴더 저장
더 이상 브라우저 다운로드 바를 클릭하거나 무겁게 ZIP 파일로 패킹할 필요가 없습니다. 파일시스템 액세스 API(File System Access API) 보안 권한을 최초 한 번만 수락해 놓으시면, 다운로드 버튼을 누르는 순간 자동으로 오늘 날짜(예: `26-02-22`) 폴더를 생성해 깔끔하게 원본 파일명 그대로 다운로드해 드립니다.

### 5. 똑똑한 유틸리티 옵션들
* 여러 사진 다중 드래그 & 드롭 한 번에 추가
* 용량 % 절감 실시간 모니터링 뱃지
* 다이렉트 지정 해상도 및 비율 고정(Aspect Ratio) 변환 기능
* 품질/용량 손실 압축 매니저
* 그레이스케일, 각종 세부 옵션들을 세트로 저장(즐겨찾기)해두고 계속 탭하여 불러올 수 있는 **프로파일 북마크** 시스템

## 🛠 아키텍처 및 기술 스택
- **프레임워크:** Next.js (15+ App Router) & React 19
- **사용자 인터페이스 & 디자인:** Tailwind CSS, Lucide-React 
- **AI 추론 및 픽셀 조작:** `@huggingface/transformers`, JS HTML Canvas API
- **데이터 캐싱 메커니즘:** Zustand (Persist), 로컬 IndexedDB
- **스토리지 및 아카이빙 처리:** JSZip, File-Saver, 로컬 파일 시스템 파일 핸들링(File System Handle API)

<br>

## 🧑‍💻 로컬 설치 및 구동 방법
해당 프로젝트는 Node.js 환경에서 간편하게 빌드 및 구동을 할 수 있습니다.

```bash
# 1. 패키지 설치
npm install

# 2. 로컬 개발 환경 띄우기
npm run dev

# 터미널에 뜨는 http://localhost:3000 접속 시 실행됩니다.
```
